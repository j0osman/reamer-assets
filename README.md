# reamer-assets

Media host for [reamerlabs.com](https://reamerlabs.com) — screenshots, video, and
future blog post images/video. Kept as a separate public repo so
[reamerlabs.com's website source](https://github.com/j0osman/reamer-website) can
stay private while its large media still gets free GitHub-hosted storage and
GitHub Pages serving (no per-file size cap, correct content-type headers for
video, unlike jsDelivr/raw.githubusercontent.com).

Not a project in itself — just a public bucket for files referenced by `<img>`/
`<video>` tags on the main site, at:

```
https://j0osman.github.io/reamer-assets/assets/<filename>
```
